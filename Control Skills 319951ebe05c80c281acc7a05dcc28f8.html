Control Skills

Goals

The Control Skills mechanic is designed to:

  • Deliver Potent Crowd Control: Enable unblockable, RPG-inspired CC via Impact State’s Grab, amplifying opponent disruption in The Fighting Game’s high-fantasy battles.
  • Balance Risk and Reward: Pair high-impact hard CC (e.g., Stun) with steep Energy/Poise costs and diminishing returns, ensuring PVP fairness.
  • Enhance Tactical Depth: Support soft CC buildup (e.g., Slows) across attacks, chaining into hard CC for strategic combos in “Core Loop”’s Advantage phase.
  • Embrace Customization: Allow players to craft CC effects (physical or magical), mirroring Path of Exile 2’s skill diversity and fighting game versatility.

Overview

Control Skills are unblockable, high-cost crowd control attacks triggered by the Grab button (LT/ALT) in Impact State (RT/RMB hold), extending the Grab Category with RPG-style effects in The Fighting Game’s 3rd-person arena. As the pinnacle of Impact State’s “control” focus, they deliver hard CC—e.g., Stun (1-2s disable), Freeze (2s immobile)—while Impact’s Light Attack (RB/LMB) handles knockup/knockdown. Soft CC (e.g., Slows 30% speed, Knockbacks ~2-3m) builds via normals (no direction), command normals (e.g., forward + RB), or other attacks (e.g., cold-based Slow), contributing to a visible CC meter tied to Poise. Repeated Control Skills trigger diminishing returns (e.g., ~50% effect duration after 2 uses in ~10s) to prevent stunlocks, with repositioning CC (Knockback, Pull) exempt. Costs scale with effect strength (15-30% Energy, ~1-2 Poise), customizable per skill crafting. Control Skills shine in setups—e.g., Stun into Pre-Jump knockup (“Knockup Attacks”) or Pull into okizeme (“Knockdown to Prone”)—with animations (e.g., lightning Stun, wrestling Disarm) reflecting player-chosen physical or magical flair. HUD tags (e.g., “CONTROL: STUN”) and a Poise/CC meter ensure readability, balancing The Fighting Game’s chaotic PVP.Key Features

  • Impact State Control Skills: Grab (LT/ALT) in Impact State delivers hard CC—e.g., Stun (1-2s, +10f advantage), Freeze (2s), Confusion (2s input scramble), Polymorph (2s model swap), Blind (~2s vision blur)—unblockable, costs ~20-30% Energy + ~1-2 Poise, HUD “CONTROL: [EFFECT].”
  • Soft CC Buildup: Normals (e.g., RB) and command normals (e.g., forward + RB) apply soft CC—e.g., Slow (30%, ~3s), Tether (3m leash, 3s), Knockback (2-3m), Pull (2-3m), Disarm (3s no attacks), Root (2s, tentative)—costs ~10-15% Energy, builds CC meter (10-20 points per hit).
  • CC Meter: Visible Poise/CC bar (100 points) tracks buildup—hard CC triggers at ~80-100 (e.g., Slow buildup to Freeze), decays ~10 points/s after ~5s inaction. Diminishing returns (50% duration after 2 uses in ~10s, resets ~15s) limit hard CC; repositioning CC exempt.
  • Non-Control CC: Other attacks (e.g., Impact Light knockup ~20f, Grab Pull ~2m) apply soft CC, ~50% buildup contribution (e.g., ~5-10 points), weaker than Control Skills (e.g., Root ~3s vs. ~2s).
  • Input Logic: Impact Control Skills use Grab (LT/ALT), normals/command normals use Light/Heavy (RB/RT) across states—e.g., Base forward + RB = Slow, Impact back + LT = Stun.
  • Visual Distinction: Physical (e.g., wrestling Disarm) or magical (e.g., arcane Polymorph) animations, customizable—e.g., Fire Stun adds flames. HUD tags and Poise/CC bar glow signal buildup.
  • Setup Potential: Hard CC enables combos—e.g., Stun (+10f) to Pre-Jump knockup, Pull to okizeme. Soft CC sets up—e.g., Slow to Impact knockup.

Challenges and Solutions

  • Input Overlap with Grabs: Grab (LT/ALT) for Control Skills and standard grabs risks confusion—e.g., expecting grapple in Impact State.
    • Solution: Lock Control Skills to Impact State’s Grab, standard grabs to Base State—HUD tag (“CONTROL” vs. “GRAB”) and distinct visuals (e.g., magical aura vs. physical slam) clarify intent; test input recognition.
  • CC Balance: Hard CC (e.g., ~2s Stun) could stunlock without tight diminishing returns, while soft CC spam (e.g., Slows) might overwhelm.
    • Solution: Cap hard CC at 2s, diminishing returns at ~50% duration after 2 uses in ~10s—e.g., Stun ~2s, then ~1s, then ~0.5s. Soft CC limited by Energy (10-15%) and Poise/CC meter (~100 points); test PVP comeback potential.
  • Buildup Ambiguity: Visible Poise/CC meter needs clear thresholds—e.g., when does Slow trigger Freeze?—and non-Control CC’s role is vague.
    • Solution: Set 80-100 points for hard CC, ~10-20 per soft CC, ~5-10 for non-Control CC—e.g., ~5 Slows (15 points each) trigger Freeze. Meter glows at 75% threshold; prototype buildup speed vs. decay (10 points/s).
  • Effect Redundancy: Knockup, Knockdown, Knockback in Control Skills overlap with “Knockup Attacks,” “Knockdown to Prone,” and Impact Light—risking dilution.
    • Solution: Control Skill CC is stronger—e.g., Knockup 30f vs. ~20f Impact Light, Pull ~3m vs. ~2m Grab. Non-Control CC weaker (50% duration/effect); test visual/functional distinction (e.g., magical Knockback vs. physical).

Suggestions for Prototyping

  • CC Timing: Test hard CC durations—e.g., Stun 1-2s (+10f), Freeze ~2s—vs. soft CC—e.g., Slow ~30% for ~3s, Root ~2-3s. Adjust diminishing returns (50% after 2 uses in ~10s) for PVP fun (e.g., ~2-3s max disable).
  • Poise/CC Meter: Prototype visible meter (100 points)—e.g., Slow ~15 points, Impact Knockup ~10 points, hard CC at ~80-100. Test decay (10 points/s after 5s) and glow (75% threshold) for readability.
  • Resource Costs: Start at ~10-15% Energy + ~1 Poise (soft CC), ~20-30% Energy + ~1-2 Poise (hard CC)—tweak via customization (e.g., +1s Stun for +5% Energy). Measure spam vs. comeback potential.
  • Setup Chains: Test combos—e.g., Stun to Pre-Jump knockup (3-5 air hits), Pull to okizeme (15f meaty). Ensure soft CC (e.g., Slow) boosts hard CC triggers (~80 points).
  • Visuals: Trial physical (e.g., wrestling Disarm) vs. magical (e.g., lightning Stun) animations—e.g., Fire Stun adds ~5% DoT for +5% Energy. Sync HUD tags (“CONTROL: STUN”) with Poise/CC glow.

Conclusion

Control Skills anchor The Fighting Game’s crowd control, delivering unblockable, RPG-inspired hard CC (e.g., Stun, Freeze) via Impact State’s Grab (LT/ALT), complemented by soft CC buildup (e.g., Slows, Pulls) across normals and other attacks. High Energy (20-30%) and Poise (1-2) costs, paired with diminishing returns (50% duration after 2 uses), ensure PVP balance, while a Poise/CC meter (100 points) tracks buildup for strategic depth. Stronger than non-Control CC (e.g., ~30f Knockup vs. ~20f Impact Light), they fuel “Core Loop”’s Advantage phase—e.g., Stun to knockup, Pull to okizeme—with customizable physical or magical flair. Prototyping will refine durations, buildup, and visuals, cementing Control Skills as a dynamic, high-fantasy disruptor alongside grabs and knockups.
